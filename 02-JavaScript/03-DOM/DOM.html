DOM - Document Object Model

DOM представляет любую HTML-страницу как сложный объект, который агрегирует в себя другие объекты - элементы страницы. DOM объекиняет все элементы страницы в дерево обектов, корневым объектом которого является объект 'document'. Этот объект всегда неявно создается в любом JavaScript-коде.
